# Advanced Network Scanner

## 📌 Overview
Advanced Network Scanner is a Python-based cybersecurity project designed to scan target systems and identify open or closed ports. This project demonstrates basic network reconnaissance and port scanning techniques used in cybersecurity and SOC environments.

---

# 🛠️ Tools & Technologies
- Python
- Socket Programming
- Colorama
- Kali Linux

---

# 🔍 Features
- Target IP/Domain Scanning
- Open Port Detection
- Closed Port Detection
- Fast TCP Port Scanning
- Colored Terminal Output
- Beginner-Friendly Network Reconnaissance
